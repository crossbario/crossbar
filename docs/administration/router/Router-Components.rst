Router Components
=================

*Routers* can run WAMP application components written in Python
*side-by-side*, i.e. within the same system process.

Here's an example configuration:

.. code:: javascript

    {
       "controller": {
       },
       "workers": [
          {
             "type": "router",
             "options": {
                "pythonpath": [".."]
             },
             "components": [
                {
                   "type": "class",
                   "realm": "realm1",
                   "classname": "hello.MySession"
                }
             ],
             // ... rest of router configuration ...
          }
       ]
    }
