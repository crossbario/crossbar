Example
-------

Here is an example where two subpaths are collected on a Path Service
which serves as a kind of folder:

.. code:: javascript

    {
       "workers": [
          {
             "type": "router",
             "transports": [
                {
                   "type": "web",
                   "endpoint": {
                      "type": "tcp",
                      "port": 8080
                   },
                   "paths": {
                      "/": {
                         "type": "static",
                         "directory": "../web"
                      },
                      "myfolder": {
                         "type": "path"
                         "paths": {
                            "download1": {
                                "type": "static",
                                "directory": "/tmp"
                             },
                             "download2": {
                                "type": "static",
                                "directory": "/data/tmp"
                             }
                         }
                      }
                   }
                }
             ]
          }
       ]
    }
